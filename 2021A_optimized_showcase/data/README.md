# data 目录说明

此目录用于存放输入数据，默认约定如下：

- `附件1.csv`：主索节点坐标（推荐）
- 其他命名但包含 `附件1` 字样的 CSV 也会被自动识别

## 数据格式建议

推荐列结构：

- `node, x, y, z`

兼容结构：

- 只要能提取到 3 列数值型坐标，系统可自动识别。

## 无真实数据时的行为

若检测不到 `附件1.csv`，系统会自动使用合成节点云（可复现，固定随机种子），
以保证 GIF 和仪表板仍可生成。

## 你导出到 GitHub 的建议

- 可提交脱敏后的示例 CSV。
- 若数据涉密，仅提交 `outputs/` 结果图和 `summary.json`。
