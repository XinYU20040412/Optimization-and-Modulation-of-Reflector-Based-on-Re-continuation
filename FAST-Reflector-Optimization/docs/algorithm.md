# Algorithm Details

## 1. 空间旋转与坐标统一

问题二原始脚本使用两步旋转矩阵进行坐标变换。Python 版本在 code/python/src/q2_geometry.py 中保留同构矩阵形式，先转到局部坐标系，再把优化后的节点坐标逆变换回全局坐标系。

## 2. 降维平面化

将反射面节点按极径映射到平面剖面，使用角度变量 theta 建模每个节点对应的伸缩量。这样可把 3D 控制量降到 1D 曲线拟合问题。

## 3. 遗传优化目标

优化变量：delta_r = [顶点伸缩量, 各离散角度伸缩量]

目标函数：最小化调控后工作曲面与理想抛物面之间的 RMSE。

Python 版本在 code/python/src/q2_ga.py 中实现了：

1. 初始化
2. 变异
3. 交叉
4. 劣个体替换
5. 收敛曲线记录

## 4. 插值拟合与连续控制

遗传算法输出离散角度上的伸缩量，再通过 4 次多项式拟合得到连续控制函数，用于批量计算全部工作口径内节点的伸缩量。

## 5. 问题三反射命中率计算

在 code/python/src/ray_models.py 中：

1. 使用工作抛物面和基准球面的剖面方程。
2. 计算每个采样点的切线斜率。
3. 由反射几何计算落点偏移。
4. 判断是否命中接收窗口，统计接收比。

## 6. 已修复关键 Bug

1. MATLAB 原逻辑中 y 分量径向归一化分母写错，导致坐标偏移异常。
2. MATLAB 原逻辑中回到全局坐标时误用了未调整坐标，导致结果覆盖优化效果。

这两个问题已在 Python 化实现中修复并加入注释。
