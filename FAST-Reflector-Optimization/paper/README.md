# Paper Overview

- FAST_Paper.pdf: 论文主文件（用于主页一键跳转）。
- 基于重连续化的反射镜优化调控.pdf: 原始论文文件备份。
- 附件4(1).xlsx: 论文原始结果附件。

## 论文摘要

本文围绕 FAST 射电望远镜反射面实时调控问题，建立了从空间坐标旋转、降维平面化到反射面伸缩优化的完整方法链。通过遗传算法最小化工作面与目标抛物面的拟合误差，并结合反射光路命中判据计算接收比，给出可复现实验流程与结果。

## Keywords

FAST, Reflector Control, Dimensionality Reduction, Genetic Algorithm, RMSE Minimization, Optical Path

## 主要结论

1. 工作抛物面在有效口径区域具备更高接收效率。
2. 通过参数化控制与优化，接收比可稳定提升到约 12.21%。
3. 相比基准球面接收比约 5.45%，整体提升约 124%。

## 创新点

1. 将空间几何调控问题转化为可计算的平面优化问题。
2. 在节点伸缩量求解中引入遗传优化并输出收敛过程。
3. 建立了从算法到 GitHub 可视化资产的工程化自动产物链。
